AIO Weather

AIO Weather keeps you informed about conditions including temperature, rain, snow, and wind.you can check the forecast by the hour for the next 6 days.

Project Demo Github

Features

Current Weather

The Current Weather feature provides users with up-to-the-minute information about the weather conditions at their selected location.
It displays the current temperature, allowing users to quickly assess the current atmospheric conditions.
Users can find out how fast the wind is blowing and from which direction.
Information about the level of humidity helps users understand how much moisture is in the air.

24h Weather

This feature offers a detailed breakdown of the weather throughout the day, providing users with forecasts for each hour.
Users can track how temperatures will change during different parts of the day.
Utilize visual icons to represent different weather conditions, making it easy for users to scan the forecast.

7 Day's Weather

This feature gives users a broader perspective by providing a summary of the upcoming week's weather.
Displaying the expected high and low temperatures for each day helps users plan for temperature variations.
Utilize visual icons to represent different weather conditions, making it easy for users to scan the forecast.

About

- How we Get the Idea of this project?

As we went through the ALX curriculum, we learned about how APIs work. When our project started, we thought about using what we learned about APIs and HTTP requests. We found that a Weather API is a popular and free option that we can use. We also realized that mastering APIs and HTTP requests would be a good practice, so we decided to include it in our project.
